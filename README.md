Open Marketplace Website
A scalable, AI-ready full-stack marketplace platform where users can buy, sell, and manage products seamlessly.

🚀 Overview
Open Marketplace is a modern e-commerce web application that enables users to:
🛍️ Browse and search products
🧾 List products for sale
🔐 Secure authentication & authorization
💳 Add to cart and checkout
📦 Manage orders and inventory
📊 Admin dashboard for marketplace control
